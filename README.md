iOS App -> This is an individual work

### 1. Live matches
### 2. Upcoming matches with timer
### 3. Recently matches
### 4. Match Information
### 5. Team Squad
### 6. Scoreboard
### 7. Player Search
### 8. Player Details
### 9. Team Ranking
### 10. League wise and date wise fixture
